
# Моделирование работы парка беспилотных такси

Учебный проект по многопоточности на Java.
Система моделирует взаимодействие диспетчера, такси и генератора клиентских запросов.

---

## Описание функционала

* Генерация клиентских запросов (координаты и время создания).
* Потокобезопасная очередь заказов (`BlockingQueue`).
* Диспетчер, который:

  * получает заказы из очереди,
  * выбирает ближайшее свободное такси,
  * назначает заказ.
* Несколько такси, каждое работает в отдельном потоке:

  * подъезжает к клиенту,
  * перевозит его до точки назначения,
  * уведомляет диспетчера об окончании поездки.

---

## Структура проекта

```
master.java - точка входа
clientreq.java - объект заказа
taxi.java - поток моделирующий работу такси
dispatc.java - распределение заказов по такси
clientgenerator.java - генератор новых запросов
```

---

## Как запустить

### 1. Компиляция

В терминале, находясь в папке проекта:

```bash
javac *.java
```

Появление `.class` файлов означает успешную компиляцию.

### 2. Запуск

```bash
java master
```

В консоли начнётся вывод логов такси, диспетчера и генератора.

---

## Логика работы потоков

* `clientgenerator` - периодически создаёт новые заказы.
* `dispatc` -следит за очередью заказов и раздаёт их свободным такси.
* `taxi` - обрабатывает назначенный заказ, имитируя движение задержками (`Thread.sleep()`).

---

## Остановка симуляции

В классе `master` симуляция выполняется фиксированное время (по умолчанию 20 секунд), затем аккуратно завершает все потоки.
